# Neutral-Class Noise Rate Findings

This report will be regenerated by:

```bash
python src/data/noise_rate_analysis.py \
  --manual-labels results/manual_labels_750.csv \
  --output-json results/true_noise_rate.json \
  --output-md docs/reports/NOISE_RATE_FINDINGS.md
```

## Current Status

The analysis script is implemented, but the empirical noise-rate findings require the manual-labeling pilot to be completed first. Once `results/manual_labels_750.csv` contains labels, the script will populate this report with:

- Estimated true-neutral rate among 3-star reviews.
- Estimated 3-star label-noise rate.
- Share of 3-star reviews that are textually negative.
- Share of 3-star reviews that are textually positive.
- Cleanlab-flagged sample validation.
- Original-label vs manual-label confusion matrix.
- Cohen's kappa between star-derived labels and manual text labels.
